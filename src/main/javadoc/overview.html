Taxi App Backend

Backend desarrollado en Spring Boot para la predicción de precios de viajes de taxi mediante integración con un servicio externo de Machine Learning.

Arquitectura

La aplicación sigue una arquitectura hexagonal (Ports & Adapters), separando claramente la lógica de negocio de la infraestructura.

  • Domain: Entidades del sistema (Trip).
  • Application: Casos de uso y puertos de entrada/salida.
  • Infrastructure: Controladores REST, cliente HTTP, repositorios JPA y configuración.

Flujo Principal

  1. El cliente envía un POST a /prediction.
  2. El controlador valida los datos (TripRequest).
  3. El UseCase solicita la predicción al servicio ML externo.
  4. Se crea un objeto Trip con el precio estimado.
  5. El viaje se persiste en base de datos.
  6. Se devuelve la respuesta al cliente.

Tecnologías Utilizadas

  • Spring Boot
  • Spring Web
  • Spring Data JPA
  • RestTemplate
  • Hibernate Validator
  • PostgreSQL
  • JUnit 5 + Mockito
  • JaCoCo (100% code coverage)

Calidad y Testing

La aplicación cuenta con pruebas unitarias y de integración, alcanzando un 100% de cobertura de código mediante JaCoCo.

Configuración

La URL del servicio de Machine Learning se externaliza mediante MlServiceProperties, permitiendo configuración por entorno.
